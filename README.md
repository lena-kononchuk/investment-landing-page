# Investment Landing Page

Modern landing page for investment company built with Gulp automation.

## Features

- Responsive multi-section layout
- Registration form with validation
- Investment calculator with range slider
- Partners/shareholders showcase
- Swiper carousel for testimonials
- Gulp build pipeline (minification, optimization)

## Tech Stack

- HTML5, CSS3 (compiled from LESS)
- Vanilla JavaScript
- Bootstrap 5
- Gulp.js (task automation)
- Swiper.js (carousels)
- Ion RangeSlider

## Structure

```
├── section/          # Modular HTML sections
├── js/              # JavaScript modules
├── less/            # LESS stylesheets
├── img/             # Images
├── build/           # Production build (generated)
└── gulpfile.js      # Gulp tasks
```

## Installation

```bash
npm install
```

## Development

```bash
# Build for production
gulp build

# Watch files for changes
gulp watch
```

## Project Purpose

Trial task showcasing frontend development skills including form validation, build tooling, and responsive design.
